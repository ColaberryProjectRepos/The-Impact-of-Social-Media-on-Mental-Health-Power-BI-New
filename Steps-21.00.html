Social Media Platforms Chart 21

1. In order to create this visual, you will need to download the "WordCloud" visual. Click on the ellipsis at the bottom of the list of visuals in the "Visualizations" panel, then select "Get more visuals". Once the Power BI visuals screen appears, type "Word Cloud" in the search bar. Download the one that is made by Microsoft. It should look like the screenshot below:



2. Next, Copy and paste the previously created visual, which is the "KPI scores by Relationship" chart, and then change the visual to the Word Cloud.
3. Now add the data. Unselect any previous data fields and then select "Social Media Platforms" for the category field and "Number of People" for the Values field.
4. Next, format the visual. Under "Format Visual" make the following changes:
Data colors: Switch show all to "on" and then make the colors the following:
YouTube: #000000
Discord: #6B007B
Snapchat: #FFD300
Pinterest: #EC8FCA
Twitter: #70BBFF
Reddit: #EE0101
Switch from "Visual" to "General" and make the following changes:
Properties:
Size: Height: 318 and Width: 325
Position: Horizontal: 954 and Vertical: 70
Title:
Text: Social Media Platforms

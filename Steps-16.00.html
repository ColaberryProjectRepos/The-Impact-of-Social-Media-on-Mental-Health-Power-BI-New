Creating KPI Scores by Generation Chart 16

1. Select "Line and Clustered Column Chart" from the "Visualizations" panel on the right side of the report to insert it. For the data, select "Age Generation" for the X-Axis, then click on that column again so that we can sort it. Towards the top under "Column tools", select "Sort by column" and choose "Generation sorting". Next, add "Number of People" for the column y-axis, and for the line y-axis add the following: " Average Depression Frequency", "Distractibility Rating Index", "Validation Seeking Score", "Focus Retention Score", and "Sleep Issues Frequency Index"
2. Now that we have the correct chart and data, we will properly set it up. The First thing to do will be to abbreviate all of the data values. We will start with with the column y-axis value "Number of People". Right click on the name and then select "Rename for this visual" then type "Ppl" for the new name. Repeat these steps for all of the Line y-axis values as well. The names should be changed to the following:
"Average Depression Frequency" = "DF Score",
"Distractibility Rating Index" = "DR Score" ,
"Validation Seeking Score" = "V Score",
"Focus Retention Score" = "FR Score",
"Sleep Issues Frequency Index" = "SI Score"
Now Switch to "Format Visual", towards the top of the "Visualizations" panel and make the following changes:
X-Axis:
Values:
Font: Times New Roman, size 9, color: #000000
Title: "On"
Title Text: "Age Generation"
Font: Times New Roman, size 12, color: #000000
y-axis:
Values: "On"
Title: Switch to "Off"
Font: Times New Roman, size 9, color: #605E5C
Secondary y-axis:
Values: "On"
Columns:
Color: #666666
Lines:
Colors: SI Score: #73361C, DR Score: #FD8915, FR Score: #9E00F9, V Score: #F9008D, DF Score: #12239E
Now switch From "Visual" to "General"
Properties:
Size: Height: 318 and width: 418
Position: Horizontal: 200 and Vertical: 70
Title:
Text: "KPI Scores by Generation"
Font: Times New Roman, size 12
Text color: #FFFFFF
Background color: #0D6ABF
Alignment: center
Effects:
Background: "On"
Color: #FFFFFF
Transparency: 15%
Visual Border: "On"
Color: #CCCCCC
Finally, click on the ellipsis at the top right hand corner of the chart you just created and sort axis by "Age Generation". You have now successfully created the first chart.

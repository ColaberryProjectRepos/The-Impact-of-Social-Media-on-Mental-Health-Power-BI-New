Importing Data 9


1. Click on the red button and download the attached zip file. This contains the dataset and images for this project.

2. Open Power BI desktop, select "Get Data" towards the top right and then select "Excel Workbook." Once you are accessing your files, switch from "Excel files" in the bottom right to "All files" so that you can access the zip file. Right click on the zip file and choose "open." Then select the excel file "SocialMediaAndMentalHealth" from the 4 items in the zip file.

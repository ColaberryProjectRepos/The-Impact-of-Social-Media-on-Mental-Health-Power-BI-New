Creating Measures 11

We will now be creating the measures that will be used in the project. In order to create the first measure, click on "Tables Tools" from the top banner. Once you click on that, enter the measure name and then the DAX formula for the measure. For the first measure, copy and paste this DAX code: "Average Depression Frequency = AVERAGE('SocialMediaAndMentalHealth'[Depression Frequency])/5" Towards the top you should see a "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage"and change the number of decimal places from 2 to 0. Then click the little check box to the left of the code you just entered. Do the same steps for each of these DAX formulas to create the rest of the measures:
1. Distractibility Rating Index = AVERAGE('SocialMediaAndMentalHealth'[Distractibility Level])/5. Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.
2. Focus Retention Score = (SUM('SocialMediaAndMentalHealth'[Difficulty to concentrate]) + SUM('SocialMediaAndMentalHealth'[Restlessness Level])) / COUNTROWS('SocialMediaAndMentalHealth')/10. Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.
3.Validation Seeking Score = AVERAGE('SocialMediaAndMentalHealth'[Validation Seeking Frequency on social media])/5. Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.
4. Sleep Issues Frequency Index = AVERAGE('SocialMediaAndMentalHealth'[Sleep Issues Frequency])/5. Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.
5. Last Refresh Date = NOW()
6. Number of People = COUNT('SocialMediaAndMentalHealth'[Gender])
7. Total Females =IF(ISFILTERED('SocialMediaAndMentalHealth'[Gender]), CALCULATE(COUNTROWS('SocialMediaAndMentalHealth'), 'SocialMediaAndMentalHealth' [Gender] = "Female", 'SocialMediaAndMentalHealth' [Gender] = SELECTEDVALUE('SocialMediaAndMentalHealth' [Gender]))/[Number of People], CALCULATE(COUNTROWS('SocialMediaAndMentalHealth'), 'SocialMediaAndMentalHealth' [Gender] = "Female")/[Number of People]). Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.

8. Total Males = IF(ISFILTERED('SocialMediaAndMentalHealth'[Gender]), CALCULATE(COUNTROWS('SocialMediaAndMentalHealth'), 'SocialMediaAndMentalHealth' [Gender] = "Male", 'SocialMediaAndMentalHealth' [Gender] = SELECTEDVALUE('SocialMediaAndMentalHealth' [Gender]))/[Number of People], CALCULATE(COUNTROWS('SocialMediaAndMentalHealth'), 'SocialMediaAndMentalHealth' [Gender] = "Male")/[Number of People]). Open the "Format" drop down box that is set to "General" as a default. Open the dropdown box and select "Percentage" and change the number of decimal places from 2 to 0.
